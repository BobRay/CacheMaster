CacheMaster Extra for MODx Revolution
=======================================

**Author:** Bob Ray <http://bobsguides.com> [Bob's Guides](http://bobsguides.com)

Documentation is available at [Bob's Guides](http://bobsguides.com/cachemaster-tutorial.html)

CacheMaster speeds up page-loads at your site by allowing you to clear the site cache for a single
Resource when you save it, rather than the whole Resource cache.
